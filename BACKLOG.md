# Беклог

## Високий пріоритет

### Режим запису конференцій
- Запис мікрофона + системного аудіо (loopback)
- Злиття двох аудіо потоків в один файл
- Збереження аудіо файлів на диск
- Розпізнавання мовців (speaker diarization)
- Транскрипція з анотацією хто що сказав
- Детальний опис: [docs/backlog/conference-recording.md](docs/backlog/conference-recording.md)

### Глобальні гарячі клавіші
- Запуск/зупинка запису з будь-якої програми
- Налаштування комбінації клавіш у конфігу
- Потребує: `global-hotkey` або D-Bus інтеграцію

## Середній пріоритет

### Вставка в активне вікно
- Після транскрипції вставити текст у поточну програму
- Симуляція Ctrl+V або використання X11/Wayland API

### Детекція тиші
- Автоматична зупинка запису після паузи
- Налаштування порогу тиші та тривалості

### Експорт історії
- Експорт у текстовий файл
- Фільтрація по даті

### Сповіщення
- Desktop notification після завершення транскрипції
- Опціонально через `notify-rust`

## Низький пріоритет

### Статистика
- Загальний час диктовок
- Кількість слів
- Графік активності

### Неперервний режим
- Безперервний запис з автоматичною сегментацією
- Для довгих диктовок

### Редагування перед копіюванням
- Можливість відредагувати текст у вікні
- Виправлення помилок розпізнавання

### Підтримка кількох мов в UI
- Переклад інтерфейсу (EN/UK)
- Gettext або fluent

### Покращення аудіо
- Вибір пристрою введення
- Налаштування якості/формату
