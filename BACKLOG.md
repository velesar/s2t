# Беклог

## P2 — Середній пріоритет

### Редагування перед копіюванням
- Можливість відредагувати текст у вікні
- Виправлення помилок розпізнавання

### Покращення аудіо
Вибір пристроїв та налаштування якості.

**Режим диктовки (CPAL):**
- Вибір мікрофона зі списку `host.input_devices()`
- Налаштування sample rate (16kHz для Whisper)
- Gain/volume control

**Режим конференції (CPAL + PipeWire):**
- Мікрофон: вибір через CPAL (як вище)
- Loopback: вибір monitor source через PipeWire
  - Auto-detect default sink monitor
  - Або вибір вручну (`pactl list sources`)
- Окремий gain для кожного джерела

**UI:**
- Dropdown для вибору мікрофона
- Dropdown для вибору loopback (в режимі конференції)
- Індикатор рівня сигналу (вже є `get_amplitude()`)
- Test recording button
